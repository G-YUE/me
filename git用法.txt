1.在目录文件夹下运行git。输入git init 创建一个空的仓库。
2.将文件放入目录下。输入git add 文件名。或者git add . 将所有文件添加到仓库。
3.然后运行git commit -m "此次改入的备注"将文件提交到仓库。
4.git status 可以查看当前文件是否有改动没加入仓库。
5.git log 可以查看修改历史。commit是某次改动的ID，然后git reset -hard ID或者HEAD（当前版本）HEAD^(上一个版本)HEAD^^(上上一个版本)或者HEAD~100可以返回某次版本。
6.或者git reflog 他显示的是你每一次的命令。
7.删除文件git rm 文件名
8.ssh-keygen -t rsa -C "youremail@example.com" 创建SSH KEY 创建过一次就可以了
9.git remote add origin git@github.com:michaelliao/learngit.git连接远程库。
10.git push -u origin master将本地内容推送到远程库上。
